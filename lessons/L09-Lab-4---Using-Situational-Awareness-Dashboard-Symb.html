Introduction

In this lab, you will create a sub-toolset under Training on the Graphics tab for organizing symbols. You will use the SA_ColumnChart symbol from Dashboard Tools in the Situational Awareness Library to create an overview symbol and configure it to show level and temperature information for mixers in an area. Then, you will link the symbol to an area object in the Galaxy.

Once you create one symbol and link it to an area object, you will import some prebuilt symbols and link them to other area objects. Finally, you will test the ViewApp in a preview session to verify the symbols display correctly with the auto-fill content feature.

Example Output: When complete, your ViewApp will display dashboard-style bar charts showing mixer levels and temperatures, automatically filled based on the navigation context — for example, navigating to Line1 shows Line1's mixer data.
Lab Objectives: Upon completion you will be able to create symbols on the Graphics tab, search for and embed symbols from the Situational Awareness Library, configure wizard options and custom properties, substitute strings and references, configure content types, link content to automation objects, and use the Auto-Fill content feature.

Quiz

Test Your Knowledge

1. Why do you set the ContentType property to Overview on the symbol?

Setting ContentType to Overview allows the symbol to be automatically displayed in layout panes that are configured to show Overview-type content. This is what enables the Auto-Fill feature to match the correct symbol to the correct pane during navigation.

2. What is the difference between Substitute Strings and Substitute References?

Substitute Strings replaces text content (like chart headings and axis descriptions), while Substitute References replaces tag references (like Mixer100.Level.PV → Mixer100.Temperature.PV). Both features are used to customize embedded symbols for different purposes.

3. Why must you enter a single space instead of leaving the New column empty for the "Line 1" string?

The Substitute Strings dialog does not allow empty strings in the New column. Entering a single space effectively hides the redundant chart heading since "Line 1" already displays as the chart heading from the SA_ColumnChart1 symbol.

4. What does the Auto-Fill content feature do in a ViewApp?

Auto-Fill automatically places content in layout panes based on the current navigation context. It searches for content matching the pane's configured content type and fills available panes, eliminating the need to manually configure each view for every navigation item.

5. How do you link a symbol from the Graphics tab to an automation object?

Open the automation object in the Object Editor, go to the Attributes tab, click Link Content in the Content pane, browse to the symbol on the Graphics tab using the Galaxy Browser, select it, and click OK. The symbol remains on the Graphics tab but is associated with the automation object as its content.
