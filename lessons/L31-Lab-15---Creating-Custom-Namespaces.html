Introduction

In this lab, you will create a sub-toolset under Training on the Graphics tab for storing ViewApp

namespaces. You will create a custom ViewApp namespace named Sources and configure it in

the ViewApp Namespace Editor.

In the editor, you will add three ViewApp attributes, including one named URL1 for providing the

URL for the WebBrowser App.

Then you will create a symbol with two buttons and a text box. The buttons will be configured to

display a homepage and a local HTML file in a running ViewApp. The text box can be used to

navigate to other locations, such as other web sites.

You will also create a new layout with the symbol in one pane and the WebBrowser App in another.

In the ViewApp Editor, you will assign this new layout to a pane and configure a navigation item

named WebBrowser to display it.

The following shows the running ViewApp navigated to WebBrowser with the symbol shown at the

top.

Objectives

Upon completion of this lab, you will be able to:



Create custom ViewApp namespaces and attributes



Use custom ViewApp namespaces and attributes in symbols and Apps



Use the WebBrowser App in a ViewApp



Override App properties using references
