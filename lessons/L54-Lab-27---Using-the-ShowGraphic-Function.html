Lab 27 – Using the ShowGraphic() Function

Introduction

In this lab, you will create an external content item configured to open an application help

document using the PDF viewer. Then you will create a layout and assign the external content

item to it.

You will also add a Help button to the AdditionalTools symbol and configure it to open the

application help document in a popup window at runtime.

Content workflow highlight: Showing content using the ShowGraphic() script function

Objectives

Upon completion of this lab, you will be able to:



Use the ShowGraphic() function to show content in a popup window

Create an External Content Item

First, you will create an external content item for the application help document.

1.

On the Graphics tab, in the Training \ External Content toolset, create a new external

content item and name it ApplicationHelpPDF.

2.

Double-click ApplicationHelpPDF to open it in the External Content Editor.

3.

To the right of the URL or URI field, click the ellipsis and select Browse for file.

The Open dialog box appears.

4.

Navigate to the C:\Training\Other Files folder, select Training_Application_Help.pdf, and

click Open.

The path to the file appears in the URL or URI field.

5.

For Media Type, click Suggest.

Media Type changes to application/pdf.

6.

Save and close the ApplicationHelpPDF external content item, and check it in.

Add a Help Button to the AdditionalTools Symbol

Next, you will add another button in the AdditionalTools symbol. You will configure the new button

with the ShowGraphic() function to open the application help document as a popup window.

13. On the Graphics tab, in the Training \ Symbols toolset, double-click AdditionalTools to

open it in the Graphic Editor.

14. Add a button element below the Historical Data Grid button and enter Help as the text.

The name of the button is Button1.

Now you will use alignment tools to make Button1 the same size as the other buttons.

15. With Button1 selected on the canvas, press Shift, and then select the Historical Data Grid

button.

Note: Alternatively, you can make your selections from the Elements list using CTRL + click. Notice that the Historical Data Grid button has blue selection handles, while Button1 has white selection handles. The blue selection handles indicate that the Historical Data Grid button will be the point of reference for resizing Button1.

16. Click the Make Same Size alignment tool.

Button1 becomes the same size as the Historical Data Grid button.

17. Click the drawing canvas to release the selections.

Lab 27 – Using the ShowGraphic() Function

18. Double-click Button1 and add an Action Scripts animation.

19. For Trigger type, ensure On Left/Key/Touch Up is selected.

20. For the script body, enter the following:

Note: For convenience, a text file is available in the C:\Training folder that includes the script for this lab. The name of the file is Lab 27 - Using the ShowGraphic() Function.txt. 'Script body begins...

'Use the ShowGraphic() function to display a layout named as

"UserHelp" in a window with Identity as "Help".

Dim graphicInfo as aaGraphic.GraphicInfo;

graphicInfo.Identity = "Help";

graphicInfo.GraphicName = "UserHelp";

'Simply customize the graphic size.

graphicInfo.RelativeTo = aaGraphic.RelativeTo.CustomizedWidthHeight;

graphicInfo.Width = 1200;

graphicInfo.Height = 600;

'Show graphic.

ShowGraphic( graphicInfo );

'Script body ends.

21. Click OK to close the Edit Animations dialog box.

22. Save and close the AdditionalTools symbol, and check it in.
