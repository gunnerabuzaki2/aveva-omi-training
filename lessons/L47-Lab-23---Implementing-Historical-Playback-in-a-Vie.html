Lab 23 – Implementing Historical Playback in a ViewApp

Lab 23 – Implementing Historical Playback in

a ViewApp

Introduction

In this lab, you will implement the historical playback feature in your ViewApp.

You will start by configuring the historical data playback service in your Galaxy.

Then, you will import some prebuilt symbols, including a playback control panel (shown below), a

playback status indicator, and a playback dashboard symbol.

You will update the Main layout with a slide-in pane on the bottom and assign the playback control

panel symbol to it. You will also add a small pane at the top and assign the playback status

indicator symbol to it.

Then, in the ViewApp, you will add a navigation item to access the playback dashboard symbol

and test historical playback in a preview session.

Note that the playback control panel symbol is configured such that only users with access levels

greater than 9000 can start historical playback.

Objectives

Upon completion of this lab, you will be able to:



Configure the historical data replay service in the Galaxy



Use prebuilt symbols for historical playback functions in a running ViewApp



View historical data in playback mode in a running ViewApp
