Lab 4 – Using Situational Awareness

Dashboard Symbols

Introduction

In this lab, you will create a sub-toolset under Training on the Graphics tab for organizing symbols.

You will use the SA_ColumnChart symbol from Dashboard Tools in the Situational Awareness

Library to create an overview symbol and configure it to show level and temperature information

for mixers in an area. Then, you will link the symbol to an area object in the Galaxy.

Once you create one symbol and link it to an area object, you will import some prebuilt symbols

and link them to other area objects in the Galaxy.

You will test your ViewApp in a ViewApp preview session and be able to view the symbols, such as

the one shown below.

Content workflow highlight: Linking content to an object

Objectives

Upon completion of this lab, you will be able to:



Create a symbol on the Graphics tab



Search for symbols to embed in the Graphic Editor



Embed symbols from the Situational Awareness Library



Configure embedded symbol wizard options and custom properties



Substitute strings for an embedded symbol



Substitute references for an embedded symbol



Configure the Content Type property of a symbol



Link content to instance objects



Change content name within an object



Import symbols



Use the Auto-Fill content feature to display content while navigating a ViewApp at runtime
