Lab 7 – Creating a Custom Symbol

Introduction

In this lab, you will create a custom symbol to be used as a common command panel for opening

and closing valves and starting and stopping pumps and agitators, as well as changing the speed

setpoint for an agitator.

You will create and configure custom properties, as well as add button elements to the symbol.

You will configure the button elements with Pushbutton, Element Style, and Value Display

animations using custom properties.

You will also add Show Symbol animations for the Agitator, Pump, and Valve symbols to be able to

show their command panels when they are clicked in runtime.

When testing your ViewApp, you will use the command panels to perform operations on the

equipment.

Objectives

Upon completion of this lab, you will be able to:



Create and configure custom properties in a symbol



Add Button elements and configure Pushbutton animations



Use Element Style Animations



Use Show Symbol animations



Configure the FaceplateMode property for a symbol



Configure the TreatAsIcon property for embedded symbols

properties for it. You will also add and configure buttons for the symbol.

ViewApp display to show the command panels for the symbols. The title and buttons on the

command panels will be different depending on which symbol was clicked. The colors of the

will only appear on the command panel for the agitator.
