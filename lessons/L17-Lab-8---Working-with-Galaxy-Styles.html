Lab 8 – Working with Galaxy Styles

Introduction

In this lab, you will add a Status element to the CommandPanel symbol for displaying quality and

status information at runtime. You will see how it displays Pending status in a running ViewApp.

Then you will use Styles to configure overrides for the quality and status display. You will set a font

color override to the Pending status style and see the changes in a running ViewApp.

After that, you will perform some Style management tasks, including importing additional styles

into your Galaxy, renaming styles, and then exporting them with the new names.

Finally, you will configure the TitleBarApp to display a drop-down list of the styles available in the

Galaxy. At runtime, you will make selections from the list to dynamically change the active style for

the running ViewApp.

Objectives

Upon completion of this lab, you will be able to:



Use the Status element



Override styles using Configure Galaxy Styles



Export Galaxy Styles



Import Galaxy Styles



Use the TitleBarApp to change the active style in a ViewApp at runtime

Manage Galaxy Style Libraries

In the following steps, you will import some styles into your Galaxy and rename them. Then you

will export all the styles from your Galaxy.

25. In the System Platform IDE, on the Galaxy menu, select Configure | Galaxy | Styles.

The Styles area appears.

26. In the top-right of the Styles area, click the ellipsis button and select Import.

The Import Style Libraries dialog box appears.

27. Navigate to C:\Program Files (x86)\ArchestrA\Framework\Bin\AdditionalElementStyles.

A number of predefined Galaxy styles are listed.

Update Element Styles at Runtime

In the following steps, you will enable the TitleBarApp in the Main layout to include a style libraries

button. Then, you will use the button at runtime to switch to a different style and see the impact on

the running ViewApp.

40. On the Graphics tab, in the Training \ Layouts toolset, double-click Main to open it in the

Layout Editor.

41. Select the TitleBarApp in the top pane, which is Pane4.

42. On the Properties tab, change the Content and Display Area \ Style Libraries property to

SecondaryRight.

43. Save and close the Main layout, and check it in.

44. Close the preview session.
