📖 L43: Lab 21 – Adding Trending to Graphics

Module 10 — Trends, Section 2

Pages 607–616

Objectives

Upon completion of this lab, you will be able to:

  • Use the SA_Trend symbols for real-time trending

Create Symbol

First, you create and configure a trend symbol within the $Mixer template. On the Templates tab, in the Training > Project toolset, double-click $Mixer to open it in the Object Editor. In the Content pane, click Add to create a new symbol named RealTimeTrend. Double-click RealTimeTrend to open it in the Graphic Editor. On the Toolbox tab, search for the SA_Trend_MultiPen symbol and drag it to the drawing canvas.

SA_Trend_MultiPen symbol dragged to the drawing canvas — displaying a multi-pen trend with two visible data traces (stepped and sinusoidal) and value placeholders

Configure Wizard Options

On the Properties tab, configure the following Wizard Options properties:

Property PathValuePurpose
Wizard Options > PenCount3PensEnables 3 data curves on the trend graph
Wizard Options > LabelTrueDisplays labels (A, B, C) next to each curve
Wizard Options > LabelTypeObjectNameLabels derived from the trend's object name
Wizard Options > SymbolModeAdvancedUnlocks advanced trend features
Wizard Options > EngUnitsTrueShows engineering units (EU) on the graph
Wizard Options > Floating TooltipTrueEnables hover tooltips for real-time values
Properties tab showing Wizard Options with PenCount=3Pens, Label=True, LabelType=ObjectName, SymbolMode=Advanced; the trend symbol displays three pens (A, B, C) with time range selectors and statistics panel

Configure Custom Properties

Right-click the symbol on the drawing canvas and select Custom Properties. Configure the following properties using the references shown — note that the Tagname properties should be set to reference mode:

NameDefault Value
PenAMe.Level.PV
PenA_EngUnitsMe.Level.PV.EngUnits
PenA_TagnameMe.Level.PV.Description (reference mode)
PenBMe.Temperature.PV
PenB_EngUnitsMe.Temperature.PV.EngUnits
PenB_TagnameMe.Temperature.PV.Description (reference mode)
PenCMe.Agitator.Speed.PV
PenC_EngUnitsMe.Agitator.Speed.PV.EngUnits
PenC_TagnameMe.Agitator.Speed.PV.Description (reference mode)
Edit Custom Properties dialog showing PenA, PenB, and PenC properties with their data source, engineering units, and tagname references — tagname properties use reference mode (orange icon)

Substitute Strings

Right-click the symbol and select Substitute | Substitute Strings. In the dialog, replace only three strings: change A to L (level), B to T (temperature), and C to A (agitator speed). Click OK to close the dialog and observe the updated labels in the trend symbol.

Note: Replace only the A, B, and C strings. Do not perform a "replace all" operation, as this would change other unrelated strings in the symbol.
Substitute Strings dialog showing the three replacement rows — Old A → New L, Old B → New T, Old C → New A — with red arrows highlighting the modified entries
SA_Trend_MultiPen with updated labels — L, T, A replacing the original A, B, C labels on the pen selector buttons and trend lines

Set Canvas Properties

Click the drawing canvas and on the Properties tab, change Graphic > Size to Fixed. Set both FixedWidth and FixedHeight to 320. Center the trend symbol on the canvas. Then set Runtime Behavior > ContentType to Real Time Trend. Save and close the RealTimeTrend symbol, then save and close the $Mixer template, and check it in.

Properties tab showing Graphic Size set to Fixed (320×320), and Runtime Behavior ContentType set to "Real Time Trend" — the trend symbol is centered on the fixed canvas

Update the Asset Layout

On the Graphics tab, open Asset_Layout in the Layout Editor. Select Pane2 and add a new pane to the right (Pane5). Configure Pane5: set Content Type(s) to Real Time Trend and Width to 300. Save and close the Asset_Layout, and check it in.

Asset Layout in the Layout Editor with Pane5 added to the right of Pane2 — Pane5 configured with Content Type "Real Time Trend" and Width 300

Test the ViewApp in a Preview Session

Switch to the ViewApp preview session and navigate to Mixer100. The RealTimeTrend graphic should display in the top-right pane. Above the trend, click L in the legend to select the Level pen.

Mixer100 view with RealTimeTrend showing three trend pens (L, T, A) with time selector dropdown; floating tooltip displays PV: 19.88, Max: 98.66, Min: 0.00, Avg: 49.33, EU: 1
Floating Tooltip: You must first select a pen by clicking its label (L, T, or A) to view the floating tooltip. The tooltip shows PV (current process value), Max, Min, Avg, and EU for the selected pen. The Avg value is calculated as (Max + Min) / 2, not a moving average.

You can also click the arrow at the top of the trend chart to select a different duration — for example, 5 minutes. When you are finished testing, minimize the preview session.

Key Takeaways

  • The SA_Trend_MultiPen supports up to 3 or 4 pens with configurable wizard options
  • Custom properties allow you to bind pen data sources, engineering units, and tagnames to object references
  • The Substitute Strings dialog lets you rename pen labels without editing individual text elements
  • Set the canvas ContentType to Real Time Trend for the symbol to function as a trend display
  • Layout panes must have matching Content Types to display trend content

Knowledge Check

Q1: What Wizard Options property controls the number of trend pens?

The PenCount property in Wizard Options — set it to 3Pens or 4Pens to configure the number of data curves displayed.

Q2: Why must the tagname custom properties be set to reference mode?

Reference mode allows the property to dynamically resolve to the actual tagname of the automation object at runtime, rather than using a static string value.

Q3: What ContentType must the drawing canvas be set to for trend display?

Real Time Trend — this tells the runtime to treat the symbol as a real-time trend display.

Q4: How do you view the floating tooltip on a multi-pen trend?

You must first select a pen by clicking its label (L, T, or A) at the top of the trend. Then hovering over the trend chart displays the floating tooltip with PV, Max, Min, Avg, and EU values.
