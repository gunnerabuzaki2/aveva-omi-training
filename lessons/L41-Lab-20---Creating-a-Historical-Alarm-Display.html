Lab 20 – Creating a Historical Alarm Display

Introduction

In this lab, you will use the AlarmApp to create a display for historical alarms and events. You will

also create an additional symbol with configurations to help view and organize historical alarms

and events at runtime.

When testing the historical AlarmApp display, you will see how to query logged alarms and events

from the historical alarms and events views. You will also see how to use the custom symbol to

change the AlarmApp settings at runtime.

Objectives

Upon completion of this lab, you will be able to:



Configure the AlarmApp to display historical alarms and events



Use the AlarmApp to query logged alarms and events at runtime



Customize the AlarmApp at runtime using Runtime Behavior properties
