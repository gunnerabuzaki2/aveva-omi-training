AVEVA™ Operations Management Interface 2023

Lab 12 – Creating External Content for a

ViewApp

Introduction

In this lab, you will create another sub-toolset under the Training toolset on the Graphics tab and

use it for storing external content items. Then you will create two external content items.

The first external content item will be configured for displaying an image file showing a picture of a

simulated plant layout. You will associate it with a navigation item in the ViewApp Editor, so that

when navigated to that item at runtime, the image of the simulated plant layout will display.

The second external content item will be configured for displaying a spreadsheet of information

about a selected mixer, and then linked to the $Mixer template object. You will configure the

external content item with a dynamic string, which will update based on the mixer asset name.

When navigated to a mixer at runtime, the spreadsheet with information about the selected mixer

will display.

Objectives

Upon completion of this lab, you will be able to:



Create external content items on the Graphics tab



Configure external content item with static link to content



Configure external content item with dynamic link to content



Link external content to automation objects
