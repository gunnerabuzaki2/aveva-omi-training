Lab 24 – Working with .NET Controls

Introduction

In this lab, you will import the SQL Data Grid control into your Galaxy, as an App for your ViewApp.

You will add the SQL Data Grid control to a layout and configure it to query historical data based

on the asset that the ViewApp navigates to.

Then you will associate the new layout with the Plant navigation item and add it to the tabbed pane

in the Main layout.

Objectives

Upon completion of this lab, you will be able to:



Import a .NET control as an App



Configure the control properties in a layout

Assemble Controls for Import

Before you import an external control into your Galaxy as an App, you will assemble the control

and all required files into a folder. In the following steps, you will copy files for the SQL Data Grid

control into a subfolder in C:\Training.

1.

In File Explorer, expand This PC and navigate to:

C:\Program Files (x86) \ArchestrA\Framework\FileRepository\TrainingGalaxy\Vendors\

Invensys_Systems__Inc__SQLDataGridUserCtrl.dll

2.

Select all files in the folder, right-click, and then select Copy.

3.

In File Explorer, navigate to C:\Training.

4.

Create a folder for the control files, and name it SQLDataGridApp.

5.

Open the SQLDataGridApp folder and paste the copied files into it.

6.

Close File Explorer.
